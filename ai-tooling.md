# AI-Tooling Usage Documentation

This document outlines the AI tools utilized during the development, coding, and debugging phases of this project.

### Tools Used
* **Code Generation, Debugging & Refactoring:** VS Code AI extensions, ChatGPT, and Gemini.

### What Worked Well
* **Boilerplate Code Generation:** Effectively generated corpus documents, initial repository structures and provided guidance on development best practices.
* **Debugging and Troubleshooting:** Successfully diagnosed and resolved bugs by analyzing provided error logs and stack traces.

### What Didn't Work Well
* **Context Limits:** When analyzing larger chunks of the codebase, the AI lost track of broader architectural requirements.
* **Scope Drift:** The AI occasionally lost focus on the primary task, suggesting unprompted optimizations or making unnecessary modifications to the code.
